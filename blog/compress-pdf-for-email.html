Why is my PDF so large?

The most common culprit is embedded images. When you export a presentation from PowerPoint, scan a document, or save a report with high-resolution photos, the PDF embeds those images at full resolution ‚Äî often 300 DPI or higher.

Sending PDF via WhatsApp

WhatsApp allows document attachments up to 100 MB, but very large PDFs load slowly on mobile connections and take up significant storage. For smooth sharing, aim for under 10 MB.

Use the Maximum compression setting (72 DPI) in TurboConvert. This is optimised for screen viewing ‚Äî exactly what WhatsApp recipients will do. The result typically looks identical to the original when viewed on a phone screen.

Tips for the smallest file size

  • Use Maximum compression ‚Äî 72 DPI is perfectly readable on screen and produces the smallest files. Only use higher quality if the PDF will be printed.
  • Don't compress twice. Compressing an already-compressed PDF loses quality without significant size reduction. Start from the original file.
  • Split long documents. A 100-page report can be split into sections using the Split PDF tool, then send each section as a separate attachment.
  • Convert to images if needed. If someone just needs to read a PDF and you can't get it small enough, use PDF to JPG and send the images instead.
  • Use cloud sharing for very large files. If the PDF is above 50 MB even after compression, use Google Drive or WeTransfer and share a link ‚Äî this bypasses email limits entirely.
