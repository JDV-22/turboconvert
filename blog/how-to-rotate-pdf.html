Why PDF pages end up rotated

Sideways or upside-down PDF pages most commonly happen when scanning documents ‚Äî a page fed into a scanner in the wrong orientation gets saved that way. They also occur when combining PDFs from different sources, or when exporting from software that uses a different default page orientation.

Rotation options explained

  • 90¬∞ clockwise: Rotates pages to the right. Use this if pages appear rotated 90¬∞ to the left.
  • 180¬∞: Flips pages upside down. Use this for pages that appear upside down.
  • 90¬∞ counter-clockwise: Rotates pages to the left. Use this if pages appear rotated 90¬∞ to the right.

Rotation is applied as a metadata instruction ‚Äî the actual content of each page is not re-rendered, so there is no quality loss.
